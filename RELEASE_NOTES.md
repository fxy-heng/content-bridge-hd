# Release Notes

## 0.1.0

当前版本是可演示、可扩展的多平台内容发布工作台。

### 功能

- 支持公众号、知乎、B站、小红书四个平台。
- 支持自定义平台扩展。
- 支持内容模板库。
- 支持目标受众、品牌语气和行动引导。
- 支持平台适配、校验、标题备选和 AI 提示词包。
- 支持发布准备度看板和发布策略建议。
- 支持计划发布时间和一键模拟发布。
- 支持发布日志、Markdown 导出和 JSON 工作区导入导出。

### 工程质量

- 无第三方运行依赖。
- 提供 Node.js 单元测试。
- 提供烟测脚本。
- 提供 GitHub Actions CI。
- 提供 README、架构文档、使用指南、demo 脚本和评审对照表。
