# 提交检查清单

## 仓库材料

- [ ] GitHub/Gitee 仓库创建时间在开题后。
- [ ] 仓库可公开访问，或截止后设置为公开。
- [ ] README 包含项目简介、功能、依赖、启动方式、测试方式、demo 链接。
- [ ] README 明确说明第三方依赖和原创功能部分。
- [ ] demo 视频链接已放入 README。

## 开发过程

- [ ] 每个功能通过独立 PR 合入。
- [ ] 每个 PR 只做一件事。
- [ ] PR 描述包含功能描述、实现思路、测试方式。
- [ ] commit 时间戳都在批次开始与截止时间内。
- [ ] commit 分布在整个开发周期，不是最后一天一次性导入。
- [ ] 主分支每次合并后都能运行。

## 功能演示

- [ ] 能输入标题、正文、标签和封面。
- [ ] 能选择公众号、知乎、B站、小红书。
- [ ] 能自动生成平台适配结果。
- [ ] 能展示校验提示。
- [ ] 能复制或导出平台文案。
- [ ] 能一键模拟发布。
- [ ] 能展示成功、失败或排期状态。
- [ ] 能查看发布日志。

## 测试验证

- [ ] `npm run check` 通过。
- [ ] `npm test` 通过。
- [ ] 浏览器访问本地页面无明显报错。
- [ ] 移动端宽度下页面不重叠。

## Demo 视频

- [ ] 视频有声音讲解。
- [ ] 覆盖核心模块和完整工作流。
- [ ] 展示自动适配四个平台。
- [ ] 展示模拟发布和发布日志。
- [ ] 简要说明扩展更多平台的架构。
- [ ] 视频可在线播放。
