# 隐私与离线说明

## 1. 数据存储

当前版本不向任何服务器上传数据。

以下内容保存在浏览器 LocalStorage：

- 原始草稿
- 自定义平台
- 发布日志

导出的 JSON 文件只会保存到用户本地。

## 2. AI 提示词

项目会生成 AI 改写提示词，但不会自动发送到任何 AI 服务。用户需要手动复制提示词到自己选择的工具中。

如果后续接入真实 AI API，需要在 README 和 PR 描述中说明：

- 使用的模型服务
- 上传的数据字段
- API Key 配置方式
- 用户如何关闭 AI 请求

## 3. 离线能力

项目包含 `service-worker.js` 和 `manifest.webmanifest`，在支持的浏览器中可以缓存核心静态资源。

首次访问后，基础工作台可在离线状态下继续打开。LocalStorage 中的草稿和发布日志仍然可用。

## 4. 真实发布接入前的注意事项

接入真实平台 API 后，需要补充：

- 平台账号授权说明
- token 存储和刷新策略
- 用户主动撤销授权的入口
- 发布失败和重试记录
- 数据删除方式
