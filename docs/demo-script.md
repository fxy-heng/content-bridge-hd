# Demo 视频脚本

## 1. 开场

说明痛点：创作者经常需要把同一份内容同步发布到公众号、知乎、B站、小红书，但每个平台标题长度、正文结构、标签习惯和封面要求不同，手动适配效率低且容易出错。

## 2. 输入内容

打开 ContentBridge 工作台，展示左侧原始内容编辑区。

操作：

1. 点击“填入示例”。
2. 展示标题、正文、标签和封面图地址。
3. 设置一个未来的计划发布时间。
4. 勾选四个平台。

## 3. 自动适配

点击“自动适配”，展示右侧四个平台的适配结果：

- 公众号：长文结构，有导语、小标题和结语。
- 知乎：观点问答结构，有结论、拆解和建议。
- B站：视频简介结构，有看点和时间线。
- 小红书：短句和话题标签，更适合图文笔记。

同时展示质量评分和平台校验提示。

## 4. 发布前处理

展示每个平台卡片中的能力：

1. 查看平台限制。
2. 点击“复制文案”。
3. 点击“导出 Markdown”。
4. 点击“导出 JSON”生成完整发布包。

## 5. 一键模拟发布

点击“一键模拟发布”，展示发布日志：

- 如果没有设置未来时间，状态为成功。
- 如果设置了未来时间，状态为排期。
- 如果标题或正文为空，状态为失败并显示原因。

## 6. 架构说明

切到 `docs/architecture.md` 或 README，说明平台扩展设计：

- 平台元数据 `platformMeta`
- 平台适配器 `adapters`
- 校验器 `validateAdaptedContent`
- 发布器 `publishToPlatforms`

强调新增平台只需要新增平台 key、规则、校验和测试，不需要改 UI 主流程。

## 7. 收尾

总结作品完成度：

- 已支持四个平台。
- 已覆盖输入、适配、预览、校验、排期、模拟发布和日志。
- 已提供测试和架构文档。
- 后续可接入真实平台 API 和大模型改写。
